# MONTRA Website Starter

## What this includes
- Vite-based frontend
- Runtime Firebase client config fetched from backend
- Google sign-in via Firebase Auth
- Example call to protected backend endpoint `/api/me`

## Quick start
1. Copy `.env.example` to `.env`.
2. Install dependencies:
   - `npm install`
3. Start dev server:
   - `npm run dev`

## Environment
- `VITE_BACKEND_URL` (default: `http://localhost:8080`)
